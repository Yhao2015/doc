# 参数说明

参数 说明 默认值 类型 版本
chartOption 图表配置项 {} Object 0.0.2
extraOption 扩展属性,继承echarts的option,其中xAxis,yAxis,series都是数组格式 {} Object 0.0.2
width 图表宽度 600 Number 0.0.2
height 图表高度 400 Number 0.0.2
functionList 图表事件:(click,dblclick,mousedown,mousemove,mouseup, mouseover,mouseout,globalout,contextmenu) function 0.0.28

# chartOption

参数 说明 默认值 类型 版本
backgroundColor 'dark':暗黑系,'transparent':透明色, '#fffff':白色 '' string 0.0.2
color 颜色 [] Array 0.0.2
xAxis x轴数据映射key 'name' string 0.0.2
yAxis y轴数据映射key 'value' string 0.0.2
legend 图例数据映射key 'deptName' string 0.0.2
type 图表类型 'RING_RATIO_OPTION', 'PIE', 'PIE_NIGHTINGALE', 'PIE_RING_DOUBLE', 'PIE_RING_GAUGE', 'PIE_RING' string 0.0.2
seriesData 数据 [] Array 0.0.2

参数 说明 默认值 类型 版本
borderColor 边框颜色 '' string 0.0.2
